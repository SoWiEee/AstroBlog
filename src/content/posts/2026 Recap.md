# Introduction

不知道是不是買了 Heptabase 的關係，所以越來越習慣寫日記或是 recap，它在我寫文章的時候幫了大忙，有很多素材可以串起來用，也可以用 AI 的功能幫我整理好這段時間的成長和變化。

# 114-2

三月初參加了 DEVCORE Conference，學到新的攻擊面和超酷的 WAF bypass。五月去 ZUTOMAYO 快閃店和演唱會，體驗很好，但錢包體驗不好。有打 AIS3 Pre-exam 和 Hack the Tainan，檢定考了 CPE 和 TOEIC，正在準備滲透測試證照的考試，也把推甄的資料整理到六成，只要調整一下就能繳交。

然後是學校的事情，接了資安課的助教，畢業專題的完成度很高，都我在 Solo，但對雲端服務有一點認識，而且知道有什麼設定和服務可以用。期末考周還是很煩躁甚至下大雨，雖然平行計算和微算機都是簡單的筆試，但影像處理的血球細胞偵測搞超久，還好 Claude Code 幫了大忙把 metric 飆升到 0.7 以上超給力。AI 課就比較輕鬆，八成靠詠唱就能拿分，是做用來檢測 agent 的操作序列是不是合法的，沒想到評分還不錯，但穩定性還可以改進。資安期末報告我是搶頭香，兩個組員都用 AI 複製貼上超傻眼，最後還是我親自修簡報。這學期所有課都是在第 16 周結束正式課程，剩下 2 周就自主利用，雖然我還是待滿 18 周，可能會隨便晃晃，然後把畢業專題報告整理成 TANet 的要求的論文格式就差不多完成了。

這學期也是很忙，感情的話幾乎是 0 進展，下學期可能有時間吧不知道

# Summer Vocation

暑假尾聲要選課，不過因為申請減修的關係所以只要修最低 1 學分就好，其他時間都能自己利用，好開心喔~
